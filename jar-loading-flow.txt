┌─────────────────────────────────────────────────────────────────────────────┐
│                              补充说明                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│  1. 双亲委派模型 (Parent Delegation Model)                                  │
│     - 类加载请求先委托给父加载器处理                                          │
│     - 自顶向下尝试加载，自底向上返回结果                                      │
│     - 防止类的重复加载                                                       │
│                                                                             │
│  2. JAR 文件结构                                                            │
│     META-INF/                                                               │
│       - MANIFEST.MF (入口配置)                                              │
│       - INDEX.LIST                                                          │
│     com/example/ (包目录)                                                   │
│     *.class (编译后的字节码)                                                │
│                                                                             │
│  3. 类加载时机                                                              │
│     - new 实例化                                                            │
│     - 访问静态字段/方法                                                      │
│     - 反射调用                                                              │
│     - 子类被加载前需先加载父类                                               │
│     - main 所在类                                                           │
